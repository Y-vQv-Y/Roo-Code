# ADTEC Code 3.2.0 Release Notes

This release marks the rebranding from ADTEC Code ADTEC Code to ADTEC Code and introduces Custom Modes!

---

## Feature Highlights

- **Name Change:** The extension has been renamed from ADTEC Code ADTEC Code to ADTEC Code.
- **Custom Modes:** You can now create your own custom modes (personas) for ADTEC Code beyond the built-in Code, Architect, and Ask modes. Define custom prompts and choose which tools each mode can access to create specialized assistants. Start by typing "Create a new mode for..." or visit the Prompts tab.
