# ADTEC Code 3.20.2 Release Notes (2025-06-13)

This release introduces security fixes, UI improvements, and codebase refactoring for enhanced security, user experience, and maintainability.

## Bug Fixes

*   **Security**:
    *   Force `tar-fs >=2.1.3` to fix a security vulnerability ([#4629](https://github.com/Y-vQv-Y))
    *   Limit `search_files` to workspace for improved security ([#4642](https://github.com/Y-vQv-Y))
*   **UI**:
    *   Sync `BatchDiffApproval` styling with `BatchFilePermission` for UI consistency (thanks samhvw8!) ([#4640](https://github.com/Y-vQv-Y))
    *   Add max height constraint to MCP execution response for better UX (thanks samhvw8!) ([#4644](https://github.com/Y-vQv-Y))
    *   Prevent MCP 'installed' label from being squeezed (thanks daniel-lxs!) ([#4649](https://github.com/Y-vQv-Y))
*   **Bedrock**: Reapply reasoning for bedrock with fix (thanks daniel-lxs!) ([#4645](https://github.com/Y-vQv-Y))

## Misc Improvements

*   **Context Condensing**: Allow a lower context condensing threshold (thanks SECKainersdorfer!) ([#4634](https://github.com/Y-vQv-Y))
*   **Code Quality**:
    *   Avoid type system duplication for cleaner codebase (thanks EamonNerbonne!) ([#4596](https://github.com/Y-vQv-Y))
    *   Improve PR Reviewer and Issue Fixer Rules for better guidance and targeted fixes ([#4651](https://github.com/Y-vQv-Y), [#4652](https://github.com/Y-vQv-Y))
*   **Unbound**: Add cache breakpoints for custom vertex models on Unbound (thanks pugazhendhi-m!) ([#4585](https://github.com/Y-vQv-Y))
*   **Docs**: Add docs extractor mode for comprehensive documentation extraction ([#4621](https://github.com/Y-vQv-Y))
