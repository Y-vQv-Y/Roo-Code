# ADTEC Code 3.25.16 Release Notes (2025-08-17)

This release improves provider integrations with native Ollama API support and GPT-5 chat model, enhances task history handling, and fixes several UI and stability issues.

## Provider Updates

* **Native Ollama API**: Fixed Ollama models to use native API instead of OpenAI compatibility layer for improved performance and reliability ([#7137](https://github.com/Y-vQv-Y))
* **OpenAI GPT-5 Chat**: Added support for gpt-5-chat-latest model optimized for conversational AI and non-reasoning tasks ([#7058](https://github.com/Y-vQv-Y))

## QOL Improvements

* **Enhanced Task History**: Made "enhance with task history" default to true for better context retention in conversations (thanks liwilliam2021!) ([#7140](https://github.com/Y-vQv-Y))
* **Improved Scrollbar**: Fixed scrollbar jumping in long conversations by removing the 500-message limit ([#7064](https://github.com/Y-vQv-Y))

## Bug Fixes

* **Settings UI**: Fixed allowedMaxRequests and allowedMaxCost values not displaying properly (thanks chrarnoldus!) ([#6925](https://github.com/Y-vQv-Y))
* **Provider Settings**: Fixed save button not enabling for provider dropdown and checkbox changes ([#7113](https://github.com/Y-vQv-Y))
* **XML Entity Handling**: Fixed XML entity decoding issues in diff tools that caused content mismatches ([#7108](https://github.com/Y-vQv-Y))
* **Condensing State**: Fixed condensing state not resetting when switching between tasks (thanks f14XuanLv!) ([#6922](https://github.com/Y-vQv-Y))
* **Workspace Paths**: Fixed workspace path resolution in codebase search tool (thanks NaccOll!) ([#6902](https://github.com/Y-vQv-Y))
* **Multi-folder Workspaces**: Fixed checkpoint handling for multi-folder workspace configurations (thanks NaccOll!) ([#6903](https://github.com/Y-vQv-Y))
* **Diff Processing**: Added type checking to prevent errors when processing diff content ([#6906](https://github.com/Y-vQv-Y))

## Misc Improvements

* **Task System**: Added API for resuming tasks, improved event handling, and enhanced metadata support ([#7122](https://github.com/Y-vQv-Y), [#7128](https://github.com/Y-vQv-Y), [#7117](https://github.com/Y-vQv-Y), [#7092](https://github.com/Y-vQv-Y))
* **Web Infrastructure**: Improved sitemap generation and SEO metadata (thanks elianiva, abumalick!) ([#7102](https://github.com/Y-vQv-Y), [#7096](https://github.com/Y-vQv-Y), [#6206](https://github.com/Y-vQv-Y))
* **Task Execution**: Refactored task execution system for better call stack management (thanks catrielmuller!) ([#7035](https://github.com/Y-vQv-Y))
