# ADTEC Code Privacy Notice

ADTEC Code is an internal development tool. It does not send product telemetry to ADTEC or to the original upstream project.

## Data Processing

- Project files, prompts, and terminal context stay on the local machine unless a user sends them to a configured AI provider.
- The configured AI provider receives only the prompt and context needed for the requested operation and processes that data under its own terms.
- API keys and credentials are stored in the editor's local secret storage and are sent only to the provider selected by the user.
- Commands run in the user's local development environment under the configured approval controls.

## User Control

Users can select a local model provider to keep model requests on the local network. Task history and extension state can be removed from ADTEC Code settings.

For privacy or security questions, contact the internal ADTEC Code administrator.
